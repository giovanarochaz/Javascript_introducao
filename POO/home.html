Orientação a Objetos com JavaScript

JavaScript usa objetos para organizar informações. Objetos têm ações (métodos) e características (propriedades).

Variáveis em JavaScript

Variáveis em JavaScript podem conter números, texto, valores verdadeiros/falsos, listas, objetos, nulo ou indefinido.

Propriedades

Propriedades são características associadas a um objeto. Por exemplo, um objeto "carro" pode ter propriedades como "cor" e "modelo" que podem ser acessadas usando a notação de ponto.
